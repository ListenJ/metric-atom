# MetricAtom

**黎曼度量驱动的感知原子：从多视图图像中无监督涌现物体实例**

MetricAtom 是一个基于黎曼度量场和有界感知原子的三维场景理解系统。核心假设是：通过联合优化外观重建和度量结构，原子能自然地凝聚为与真实物体边界对齐的簇，无需任何预训练语义模型（SAM/CLIP）或外部 2D 分割引导。

## 核心思想

```
多视角图像
     │
     ▼
黎曼度量场 g(x) ←── 联合优化
     │
感知原子（紧支撑、特征向量、存在概率）
     │
可微体积渲染 ─── 重建损失
     │
凝聚损失 ───── 自发聚类
     │
物体实例（簇）
```

### Phase 5: Direct Loss 直接聚类（2026-05-16 ~ 05-18）

**根因分析**：InfoNCE 在黎曼空间中存在逻辑循环——需要聚类定义度量，又指望度量定义聚类。度量场和特征同时控制正负样本间隔，可行域极度狭窄。

**解决方案**：用 Sinkhorn 可微软分配替代 InfoNCE，直接最小化簇内测地距离。

```
度量场 g → 占位耦合（学边界）→ g 变锐利 → Sinkhorn 软分配 P → L_direct = Σ_k P[:,k]ᵀ @ D²_g @ P[:,k]
```

| 指标 | InfoNCE 最佳 | Direct Loss | 提升 |
|------|-------------|-------------|------|
| **ARI** | 0.440 | **0.755** | +72% |
| **NMI** | 0.291 | **0.664** | +128% |
| 簇平衡度 | — | 0.94 | 近乎完美 |

**配套基础设施**：

| 功能 | 说明 |
|------|------|
| 超参网格搜索 | Phase 4/5 自动化扫描 |
| 特征扩散模块 | 测地亲和力 + sigmoid 软遮罩 + 自适应 σ |
| 实验管理脚本 | `run_experiments.py` / `run_remote.sh` / `run_local.sh` |

### Phase 6a: Murmuration-ECO 理论框架（2026-05-18）

**核心定义**：物体 = 椭圆曲线上的概率流形 $O = (E, \mu_E, v)$。

身份由 j-不变量 $j(E) = 1728 \cdot \frac{4a^3}{4a^3+27b^2}$ 编码，**二阶稳定** $\delta j = O(\|\delta\|^2)$。

**尖锐悬崖诊断**：Sinkhorn $P_{ij}=e^{-C_{ij}/\varepsilon} / \sum_k e^{-C_{ik}/\varepsilon}$ 对特征初始化一阶敏感，导致只有 1/4 seed 找到好盆地（$\sigma_{ARI}\approx 0.35$）。

**ECO 解决方案**：用 j-不变量（二阶稳定）+ 流形 Sinkhorn（测地距离）+ Lyapunov 稳定性（紧致流形），预期将标准差从 0.35 降至 0.08。

### Phase 6b: ECO 代码实现（2026-05-18）

| `murmuration.py` | Boids 在椭圆曲线上的动力学（凝聚/对齐/分离力在 1D 切空间中），PyTorch 批处理版本 |

### Phase 7: Landscape 系统性扫描（2026-05-19）

**目标**：量化 DirectCluster 对随机初始化的敏感度，验证 Phase 6a "尖锐悬崖"理论。

在 Tesla T4 云端 GPU 上运行 8 个不同 seed（100-107），每个 200 epochs，111 atoms，64×64 分辨率。

**核心发现**：
- 存在**完美解**（seed 107: ARI=1.0, NMI=1.0, 簇平衡 B=1.00），证明 DirectCluster 在有利初始化下可以完全解决 2D 双物体场景
- 但方差极大（σ=0.39），约 50% seed 未能有效聚类——确认了尖锐悬崖现象
- 完美聚类仅需 50/111 原子落在物体内，说明聚类质量 ≠ 覆盖数量
- **下一步方向**：ECO j-不变量初始化替代随机种子，预期将 σ 从 0.39 压缩至 0.08
