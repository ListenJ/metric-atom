# Murmuration 动力学严格分析

## Lyapunov 函数存在性 · Hartman-Grobman 稳定性 · 吸引域估计

> 方向 5：对 `src/geometry/murmuration.py` 中 Boids 动力学的严格数学分析。
> 核心结论：Murmuration 在阻尼主导条件下拥有一阶 Lyapunov 函数，
> 系统几乎全局收敛到均匀间距的晶体构型，此构型是 j-invariant 身份稳定的动力学基础。

## 符号约定

| 符号 | 含义 |
|------|------|
| `E` | 椭圆曲线: y² = x³ + ax + b, Δ = -16(4a³+27b²) ≠ 0 |
| `P_i ∈ E` | 第 i 个 agent 的位置 (x_i, y_i) |
| `v_i ∈ ℝ` | 第 i 个 agent 的速度标量（沿 T_{P_i}(E) 方向） |
| `u_P ∈ ℝ²` | P 处的单位切向量：dx:dy = 2y:3x²+a, 归一化 |
| `d_g(P,Q)` | P, Q 间的测地距离（弧长积分） |
| `log_P(Q)` | 对数映射：符号标量，正=沿 u_P 方向 |
| `N_i` | P_i 的邻居集合 {j: d_g(P_i,P_j) < R} |
| `N` | agent 总数 |

## 一、系统形式化

### 1.1 状态空间

```
S = E^N × ℝ^N

每个 agent: (P_i, v_i) ∈ E × ℝ ≅ S¹ × ℝ（当 Δ < 0 时 E(R) ≅ S¹）
```

**关键简化**：T_P(E) 是 1 维的。所有力 → 标量，状态空间 = ℝ^{2N}（在局部坐标下）。

### 1.2 连续时间动力学

将时间步长 dt → 0 得到 ODE：

```
dP_i/dt = v_i · u_i(P_i)                                    (1)
dv_i/dt = -η · v_i + F_i(P, v)                              (2)
```

其中：
- **阻尼系数**: η = (1 - damping)/dt。默认配置: η = (1-0.9)/0.1 = **1.0**
- **总力**: F_i = F_coh,i + F_align,i + F_sep,i

### 1.3 力分解（解析形式）

**凝聚 (Cohesion)**:

```
F_coh,i = α · (1/N_i) · Σ_{j∈N_i} log_{P_i}(P_j)
```

其中 `log_{P_i}(P_j)` = sign(chord·u_P) · d_g(P_i,P_j) —— 沿切向的符号距离。

对应于势能 `U_coh = ½α Σ_i (1/N_i) Σ_{j∈N_i} d_g(P_i,P_j)²`，
因为 `∇_{P_i}(沿着 u_P) [½ d_g(P_i,P_j)²] = d_g · ∂d_g/∂s = log_{P_i}(P_j)`。

**对齐 (Alignment)**:

```
F_align,i = β · (1/N_i) · Σ_{j∈N_i} v_j
```

Cucker-Smale 型的速度平均，在 1D 切空间中直接取标量平均。

**分离 (Separation)**:

```
F_sep,i = -γ · Σ_{j: d_g(P_i,P_j) < r}  sign(log_{P_i}(P_j)) / d_g(P_i,P_j)²
```

逆平方排斥。注意 `sign(log_{P_i}(P_j))` 决定方向（远离邻居）。

对应势能 `U_sep,ij ≈ γ/d_g(P_i,P_j)` 当 d < r（积分 1/r² → -1/r，加上截断）。

### 1.4 参数默认值（与代码对齐）

| 参数 | 代码 | 默认值 |
|------|------|--------|
| 凝聚权重 | `cohesion_weight` | α = 0.1 |
| 对齐权重 | `alignment_weight` | β = 0.05 |
| 分离权重 | `separation_weight` | γ = 0.2 |
| 分离半径 | `separation_radius` | r = 0.5 |
| 邻居半径 | `neighbor_radius` | R = 2.0 |
| 最大速度 | `max_speed` | v_max = 0.5 |
| 阻尼 | `damping` | 0.9 |
| 时间步长 | `dt` | 0.1 |

## 二、Lyapunov 函数构造

### 2.1 候选函数

定义 **总能量** V: S → ℝ:

```
V(s, v) = T(v) + U_coh(s) + U_sep(s)                          (3)

T(v)    = ½ Σ_{i=1}^N v_i²                                   (动能)
U_coh   = ½α Σ_i (1/N_i) Σ_{j∈N_i} d_g(P_i,P_j)²             (凝聚势)
U_sep   = γ Σ_{i,j: d_g(P_i,P_j) < r} [r - d_g(P_i,P_j)]²    (分离势)
```

> **分离势选择说明**: 使用 `[r - d]²` 而非 `1/d` 因为前者的梯度正是 `-2(r-d) · sign(log_P(Q))`，
> 在 d → r 处力→0，避免硬截断。当 d ≪ r 时，力≈-2r/d²（逆平方排斥），与代码行为定性一致。

### 2.2 能量耗散定理

**定理 2.1 (Lyapunov 衰减)**。当 η > β 时：

```
dV/dt = -Σ_i v_i² · (η - β · c_i) - Σ_i β · c'_i · v_i · (v_i - v̄_{N_i})² ≤ 0
```

其中
- `c_i ∈ [0, 1]` 是与邻居数相关的系数（`c_i = 1/|N_i|` 的变体，≤ 1）
- `c'_i ∈ [0, 1]` 是二次速度方差项的系数

特例：当 η ≥ β（即 1.0 ≥ 0.05）时，成立**严格耗散**：

```
dV/dt = -(η - β) · Σ_i v_i²  -  Σ_i  β/|N_i| · Σ_{j∈N_i} (v_i - v_j)²  +  O(∂N/∂t)
                                                                      ≤  0
```

**证明**:

```
dV/dt = Σ_i v_i · dv_i/dt  +  Σ_i (∂U/∂P_i) · dP_i/dt

dP_i/dt = v_i · u_i

∂U_coh/∂P_i (沿切向) = -α/N_i · Σ_{j∈N_i} log_{P_i}(P_j) = -F_coh,i

∂U_sep/∂P_i (沿切向) = -F_sep,i（由势能定义保证）
```

因此:

```
Σ_i (∂U/∂P_i) · dP_i/dt = Σ_i (-F_coh,i - F_sep,i) · v_i        (4)
```

同时:

```
Σ_i v_i · dv_i/dt = Σ_i v_i · (-ηv_i + F_coh,i + F_align,i + F_sep,i)
                  = -η Σ_i v_i² + Σ_i v_i·F_coh,i + Σ_i v_i·F_align,i + Σ_i v_i·F_sep,i  (5)
```

(4)+(5):

```
dV/dt = -η Σ_i v_i² + Σ_i v_i·F_align,i +  [Σ_i v_i·F_coh,i - Σ_i v_i·F_coh,i]
                                            + [Σ_i v_i·F_sep,i - Σ_i v_i·F_sep,i]
      = -η Σ_i v_i² + Σ_i v_i·F_align,i                                         (6)
```

> **关键洞察**: 凝聚和分离是保守力（来自势能），它们的贡献在 dV/dt 中完美抵消。
> 唯剩对齐项和阻尼项。

对齐项展开:

```
Σ_i v_i·F_align,i = Σ_i v_i · (β/N_i Σ_{j∈N_i} v_j)
                  = β Σ_i (1/N_i) Σ_{j∈N_i} v_i v_j                             (7)
```

利用恒等式 `v_i v_j = ½(v_i² + v_j² - (v_i-v_j)²)`:

```
Σ_i v_i·F_align,i = β Σ_i (1/N_i) Σ_{j∈N_i} [v_i² + v_j² - (v_i-v_j)²]/2
                  = β Σ_i v_i² - β/2 Σ_i (1/N_i) Σ_{j∈N_i} (v_i-v_j)²          (8)
```

（第一项来自对称性 Σ_i (1/N_i) Σ_{j∈N_i} (v_i²+v_j²)/2 = Σ_i v_i²）

代入 (6):

```
dV/dt = -(η - β) Σ_i v_i²  -  β/2 Σ_i (1/N_i) Σ_{j∈N_i} (v_i-v_j)²  ≤  0   (9)
```

当 η ≥ β（默认 η=1.0, β=0.05）成立严格不等式。∎

### 2.3 能量耗散率

**推论 2.2**。设 v̄² = (1/N) Σ_i v_i² 为均方速度，则:

```
dV/dt ≤ -(η - β) · N · v̄²                                          (10)
```

当 η ≫ β（默认情况）时，能量以速率 `-0.95 · N · v̄²` 衰减。
这对应 O(exp(-(η-β)·t)) 的指数级速度衰减。

### 2.4 分离势的微分包含

由于邻居集合 N_i 在运动中是分段常数（当 agent 穿越邻居半径 R 时变化），
(9) 的形式在 N_i 变化点处有效，而 V 本身在 N_i 变化时保持连续
（因为 U_coh 和 U_sep 中的截断在边界处平滑）。所以 V 全局连续且非增。

**命题 2.3**。V 是系统的 (非严格) Lyapunov 函数。当 η > β 且存在非零速度时，V 严格下降。

## 三、平衡点分析

### 3.1 平衡条件

平衡点 (P*, v*) 满足 dP_i/dt = 0 且 dv_i/dt = 0:

```
v*_i = 0         ∀i                                            (11)
F_coh,i + F_sep,i = 0    ∀i                                     (12)
```

条件 (12) 即凝聚吸引力 = 分离排斥力，在每一个 agent 处恰好平衡。

### 3.2 均匀间距解

**命题 3.1**。设 E 的 Δ < 0（单连通分支 ≅ S¹），则存在一族平衡构型：

```
s*_i = i · L/N + s₀     (i=0,...,N-1)                          (13)
```

其中 s 是沿 S¹ 的弧长参数，L = 周长，s₀ 是任意参考原点。
这些构型是迭代对称的（在 2π/N 旋转下不变）。

**证明**: 均匀间距下，每个 P_i 的左右邻居对称分布。
在 1D 紧流行上，吸引和排斥关于每个 agent 对消：
F_coh 从两侧拉 → 对称时合力为零；F_sep 推离 → 同样对称。
由于 T_P(E) 的标量限制，力直接相加。∎

**构型数**: 有 N 个不同的这种构型（对应 (N-1) 个可能的均匀间距模式，加上镜面对称→退化）。

### 3.3 其他平衡点

- **聚类构型**: 所有 agent 聚集在一小段上 → F_sep 远强于 F_coh → 不稳定（排斥 → 扩散）
- **二分构型**: agent 分裂为两个聚团 → 内部平衡但全局不平衡 → 亚稳态（能量局部极小值，但非全局）
- **随机构型**: 力零点是孤立的 → 不稳定鞍点

**命题 3.2**。均匀间距构型 (13) 是全局能量最小的（在给定 N 和 R 下），即:

```
argmin_{(P_i)} U(P) = 均匀间距构型（或其 N-1 个旋转等价版本）
```

**证明思路**: U_coh 惩罚分散（鼓励聚集），U_sep 惩罚拥挤（鼓励分散）。
在 S¹ 上，能量泛函是凸旋转不变的 → 均匀间距是唯一（模旋转）全局极小值。
这类似 Thomson 问题在 S¹ 上的退化版（1D 约束使解显式可求）。∎

## 四、Hartman-Grobman 线性化与局部稳定性

### 4.1 Jacobian 结构

在均匀间距平衡点 s* 附近线性化。设 δs_i = s_i - s*_i, δv_i = v_i（在平衡点 v*=0）。

线性化系统:

```
d/dt [δs]   =   [0     I ] [δs]
     [δv]       [J   -ηI + βA] [δv]                            (14)
```

其中:
- `J ∈ ℝ^{N×N}`: 力的 Jacobian = ∂F/∂s|_{(s*,0)}

  在均匀间距下，J 是 **循环矩阵**（每个 agent 有相同的局部几何）。
  J_{ij} 仅依赖于 j-i 模 N 的距离。

  ```
  J_{ii} = - (α·N_i' + γ·N_i'')      (对角：自身排斥）
  J_{i,i±k} = ∂F_i/∂s_{i±k}           (取决于邻居层 k 的具体距离)
  ```

- `A ∈ ℝ^{N×N}`: 对齐邻接矩阵

  ```
  A_{ij} = 1/N_i   if j∈N_i, else 0
  在均匀间距下也是循环矩阵
  ```

### 4.2 特征值分析

因为 J 和 A 都是循环矩阵，它们共享同一组特征向量——离散 Fourier 基:

```
e^{(k)}_j = exp(2π i · k · j / N)     k = 0, ..., N-1
```

设 λ_J(k) 和 λ_A(k) 为 J 和 A 在模式 k 下的特征值:

```
λ_J(k) = Σ_{m=-(N-1)}^{N-1} J_{0,m} · exp(2π i k m / N)        (15)
λ_A(k) = Σ_{m∈N_0} (1/|N_0|) · exp(2π i k m / N)               (16)
```

系统矩阵 (14) 是 2N × 2N 的分块矩阵。每个 Fourier 模式 k 产生 2×2 子矩阵:

```
M_k = [0           1     ]
      [λ_J(k)   -η + βλ_A(k)]
```

特征方程:

```
μ² + (η - βλ_A(k)) · μ - λ_J(k) = 0                             (17)
```

**稳定性条件**: Re(μ) < 0 对所有 k 成立。

### 4.3 凝聚/分离势的凸性

**命题 4.1**。在均匀间距平衡点，J 是负半定的（所有 λ_J(k) ≤ 0）。

**物理直觉**: 均匀构型是势能 U 的极小值 → Hessian J = -∇²U 是负半定（因为力 = -∇U）。

定量：

对于局部分析，近似：
- 凝聚力 = α · δs（弹簧型，大间距时为微弱的线性吸引）
- 分离力 = γ · (r-d)/r² （d < r 时强排斥）

在均匀间距 d* 处：
- 若 d* < r: 分离主导 → ∂F/∂s 为负（稳定排斥回复力）
- 若 d* > r: 凝聚主导 → ∂F/∂s 为负（弹簧回复力）
- 无论如何，平衡点周围 δF/δs < 0 → J 负半定

计算 λ_J(k) 在模式 k=0（全局平移）: λ_J(0) = 0（平移对称性 →零模式）。

对于 k=1,...,N-1: λ_J(k) < 0 → 稳定。

**命题 4.2 (谱间隙)**。最小非零特征值（谱间隙）：

```
λ_min = min_{k>0} |λ_J(k)| ≈ (α - γ/r²·∂d/∂s|_{d=d*}) · (2πk/N)²    (18)
```

其中 ∂d/∂s|_{d=d*} 是在均匀间距 d* 处距离对弧长的导数。

典型量级估计（`N=100, L≈10, α=0.1, γ=0.2, r=0.5`）:
d* ≈ L/N = 0.1（若严格均匀）但受 R=2.0 限制，实际邻居间距 ≈ 0.5-2.0。

谱间隙 `|λ_J(1)| ≈ O(α) ≈ 0.1`。

### 4.4 完整稳定性

**定理 4.3 (局部渐近稳定性)**。设 η > β, 则均匀间距构型 (13) 是局部渐近稳定的:

```
∃ δ > 0, s.t. ||(s(0), v(0)) - (s*, 0)|| < δ ⇒ lim_{t→∞} (s(t), v(t)) = (s*, 0)
```

**证明**: 对 (17) 的每个 k 应用 Routh-Hurwitz。

```
μ² + (η - βλ_A(k))μ + (-λ_J(k)) = 0
系数: a₁ = η - βλ_A(k), a₀ = -λ_J(k)
```

由于 λ_J(k) ≤ 0（命题 4.1）→ a₀ ≥ 0。
λ_A(k) ∈ [-1, 1]（A 是双随机的 → 谱半径 = 1）→ η - βλ_A(k) ≥ η - β > 0 → a₁ > 0。

两个系数均正 → 所有根有负实部。k=0 模式给出 λ_J(0)=0, a₀=0 → μ=0 或 μ=-(η-β) < 0。
μ=0 对应全局平移自由度（LaSalle 不变性→趋于匀速平移→速度阻尼为零）。∎

## 五、吸引域估计

### 5.1 能量方法

**定理 5.1 (吸引域下界)**。设 `Ω(V₀) = {(s,v): V(s,v) ≤ V₀}` 是 V 的水平集。
若 `Ω(V₀)` 不包含其他平衡点（除了均匀间距构型），则 `Ω(V₀)` 全在被吸引域内:

```
BA ⊇ Ω(V_max safe)                                              (19)
```

其中 `V_max safe = inf{V(P): P 是 Ω 内除均匀构型外的另一平衡点}`。

**证明**: V 沿轨道严格下降（定理 2.1 + η > β），因此 `Ω(V₀)` 是前向不变的。
由 LaSalle 不变原理，所有轨道趋向最大不变集 = {均匀间距构型（及其旋转）}。
极限点唯一性由各局部极小值的孤立性保证。∎

### 5.2 安全能量界

需要找出导致非均匀平衡的能量水平。

**最危险情况**: 两个 agent 进入分离半径 r 内。此时 U_sep 急剧上升（~1/d），
但可能触发"驱逐"——agent 被弹出邻居半径 R 外，孤立化。

**命题 5.2**。安全条件:

```
d_min := min_{i≠j} d_g(P_i, P_j) > r/N_eff                      (20)
v_max_initial := max_i |v_i| ≤ v_max  (= 0.5)                    (21)
```

其中 `N_eff ≈ πR/L · N` 是典型邻居数。

当 (20) 和 (21) 满足时，系统在 Ω(V(初始)) 内不经过奇异点，
且能量单调下降 → 收敛到均匀构型。

### 5.3 吸引半径数值估计

在局部坐标下（S¹ 弧长参数化），能量:

```
V ≈ ½ Σ_i v_i² + ½α Σ_i Σ_{j∈N_i} (s_j-s_i)^2 + γ Σ_{d<r} (r-d)^2
```

在小扰动 δs 下展开（v=0 附近）:

```
V ≈ ½ Σ_i v_i² + ½ (-λ_J(1)) · Σ_i (δs_i - δs̄)²               (22)
```

**推论 5.3**。吸引域包含所有满足的初值:

```
||δs||_∞ < min(d*/2, r) 且  ||v||_∞ < v_max                   (23)
```

数值（N=100, L≈10 → d*≈0.1, r=0.5）: 位置误差 < 0.05（弧度≈5°），速度 < 0.5。

这是保守界。实际吸引域更大（因为能量壁垒高于此线性界）。

### 5.4 对齐项对吸引域的影响

β（对齐权重）在 dV/dt 中引入 `-β/2 Σ (v_i-v_j)²` 项（公式 9）。
这加速收敛（减小速度方差），但由于 β ≪ η（0.05 vs 1.0），不影响能量单调性。

对齐力不会将系统推出吸引域——它只产生沿切向的对称速度滑移，
不改变势能 U（纯相位变量），因此 Ω(V₀) 在对齐力下仍保持前向不变。

## 六、与训练动力学的联系

### 6.1 Murmuration 在训练循环中的角色

训练循环: sensing(φ) → murmuration → Sinkhorn → identity matching

```
每个 epoch:
  1. φ: features → (a,b) ECO 参数  （感知）
  2. Murmuration: P_i evolves N_murm steps  （涌现）
  3. Sinkhorn: 软聚类 assignment   （组织）
  4. L_identity = ||j(φ(f)) - j(φ(f))||²   （身份一致性）
```

原始（未 murmurate 的）ECO 点因感知噪声而随机分布于曲线上。
Murmuration 将这些点推到"自然构型"——均匀间距——使 j-invariant 计算稳定:

**Proposition 6.1**。Murmuration 后的 j-invariant 方差:

```
Var[j(P_1^*,...,P_N^*)] ≪ Var[j(P_1^0,...,P_N^0)]               (24)
```

因为均匀构型的离散采样在任何 j-独立扰动下都是小方差的。

### 6.2 j-stability 的动力学机制

j-invariant 是 **全局** 不变量——它与曲线参数 (a,b) 挂钩，不直接依赖 ECO 点在曲线上的位置。
然而，当 ECO 点杂乱分布时：

1. 某些 P_i 接近无穷远点 → 采样偏差
2. 数值误差在 geodesic_distance 计算中累积

Murmuration 消除二者：均匀分布 → 每个 P_i 贡献等权 → j 回归真实值。

### 6.3 Murmuration 的相变与训练 Phase 对应

| Murmuration 状态 | 训练 Phase | 行为 |
|---|---|---|
| 自由扩散 (v ≈ random) | Phase 1-5 (重建主导) | 原子还在学习空间布局，ECO 参数 (a,b) 大幅波动 |
| 对齐-居间 | Phase 5-6 过渡 | 原子开始稳定，(a,b) 变化减缓，murmuration 开始收敛 |
| 晶体 (v ≈ 0) | Phase 6-7 | ECO 点锁定在均匀构型上，j-invariant 稳定 |

与 `framework_audit.md` 中 Phase 7 的"双峰景观"对应:
- 低能量谷: 均匀构型（murmuration 已收敛）
- 高能量谷: 随机构型（murmuration 未收敛或陷入亚稳态）

### 6.4 初始条件敏感性解释

Phase 7 的双峰行为（`framework_audit.md` §3.3）现在可以从动力学角度理解：

- **Seed 敏感**: 不同的原子/ECO 初始化 → 不同的始解构型
- **能量壁垒**: V(s₀) 高于吸引域边界 → 陷入亚稳态（非均匀构型）→ 高 loss
- **壁垒高度**: `ΔV ≈ γ · N·(1/r_min - 1/d*)` 其中 r_min = 最小粒子间距

当初始 r_min < r/N_eff（命题 5.2 违反）时，系统高概率落入非均匀 basin。

## 七、未解决 / 需要后续探索的问题

1. **有限时间步长效应**: 连续时间 V 衰减保证在 dt > 0 时变为 `ΔV ≤ -(η-β)·Σ v²·dt + O(dt²)`。
   O(dt²) 项在 dt=0.1 时多大量级？是否会打破 Lyapunov 性质？

2. **多连通分支（Δ > 0）**: E(R) = 2 个分支。Agent 能否跃迁？
   在代码中，exp_map 用组律 + Newton 投影 → 不能跨越分支间隙（Δ > 0 时曲线断开）。
   这保证了 agent 停留在初始分支上。

3. **拓扑障碍**: S¹ 上的均匀构型是唯一的（模旋转），这意味着没有局部极小值？
   实际上，不均匀的 N 聚团（如 k 组均匀间隔的聚团）是势能的局部极小值
   → 可能形成"部分凝聚"的亚稳态构型。

4. **变曲率效应**: 实际椭圆曲线有非均匀度量 g(θ) dθ²（与均匀 S¹ 不同）。
   这改变力的空间分布 → 均匀间距 ≠ 等弧长间距 → 真正的平衡构型需要求解
   `F_coh,i + F_sep,i = 0` 考虑变 g(θ)。

5. **与 Cucker-Smale 模型的精确联系**: 本系统是 Cucker-Smale 在 S¹ 上
   加逆平方排斥和可变度量的版本。Cucker-Smale 的 flocking 定理在此处有何对应？
