## 一、提案摘要

提案建议引入**确定性特征扩散**机制：基于度量场定义的测地亲和矩阵，对原子特征进行可微的局部平滑。核心主张：

1. 零额外参数（权重由度量场 + 原子位置完全决定）
2. 全可微（梯度回传至度量场和原子位置）
3. 可替代 KMeans 初始化
4. 预期 ARI 从 0.175 → 0.5+

---

## 二、逐条审查

### ✅ 2.1 零参数设计 — 通过

亲和矩阵 $A_{ij}$ 完全由度量场 $g$ 和原子位置 $\mu_i$ 定义，扩散过程仅含超参（$\sigma, \alpha, T$），无可学习参数。这符合系统"度量场是唯一底本"的架构原则。**设计正确**。

### ⚠️ 2.2 全可微性 — 有条件通过

梯度路径为：

```
L_total ← F' (扩散后特征) ← A (亲和矩阵) ← g (度量场), μ (原子位置)
```

数学上每条路径都可微。但有两个隐藏问题：

**问题 1：硬截断 $\tau_{\max}$ 破坏梯度**

```python
mask = (d2.sqrt() < tau_max)  # 二元掩码，梯度为零
A = A * mask                  # 梯度被 mask 截断
```

截断边界处的原子对梯度贡献为零。缓解方案：用 sigmoid 软掩码替代硬截断：

```python
# 软截断（可微）
soft_mask = torch.sigmoid((tau_max - d2.sqrt()) / temperature)
A = A * soft_mask
```

**问题 2：矩阵求逆不在路径上，但 $S = D^{-1}A$ 中 $D$ 对 $A$ 求导是线性的，回传正确**。验证通过。

### ⚠️ 2.3 KMeans 可移除 — 不通过

提案称"扩散从训练伊始就在不断平滑，无需 Phase 1 结束时的显式聚类初始化"。这是**有风险的假设**。

分析如下：

- KMeans 在 epoch 250 做**全局特征分配**：2 个簇各自获得不同的特征基向量
- 扩散是**局部平滑**：特征只能沿着测地邻接图传播
- 如果初始特征全是随机噪声（无全局偏置），扩散只会局部模糊，**不会自发形成两个全局分离的簇**

**类比**：扩散好比在每个原子周围涂抹颜料，如果一开始没有两种颜色，涂抹再多也不会出现两种颜色。KMeans 的作用就是提供这两种"底色"。

**建议**：保留 KMeans 初始化，将扩散作为 KMeans 的补充（而非替代）。KMeans 提供全局结构种子，扩散负责局部细化。

### 🔴 2.4 ARI 0.175 → 0.5+ — 过度承诺

这是提案中**最弱的断言**，原因：

1. **扩散不创造新的分离信号**。它只平滑已有特征。如果两个物体间的测地边界不够清晰（即度量场未完全分开物体/背景），扩散可能把不同物体的边缘原子也平滑到一起，反而降低 ARI。
2. **没有机制防止过平滑**。如果 $\sigma$ 过大或 $T$ 过多，整个特征场趋同，ARI 归零。
3. **消融实验缺失**。提案没有提供任何 toy example 或理论分析证明扩散能提升 ARI。

**修正断言**：扩散预期能提升**特征簇内紧致度**（降低同物体原子的特征方差），但对**簇间分离**的贡献是间接的（通过更好的梯度信号帮助度量场学习）。ARI 提升幅度取决于度量场本身的学习进度。

### ✅ 2.5 与现有组件的集成 — 通过

提案描述的集成方式（渲染前扩散特征、保留低权重 InfoNCE）是合理的，不会破坏现有管线。

## 三、技术细节问题

### 3.1 中点度量 vs. 端点度量

计算测地距离时，提案建议用中点度量 $g((\mu_i + \mu_j)/2)$，而当前实现直接在端点处计算 $g(\mu_i)$ 和 $g(\mu_j)$ 后做 einsum。

当前实现实际计算的是：

$$d_{ij}^2 \approx (\mu_i - \mu_j)^\top g(\mu_i) (\mu_i - \mu_j)$$

这是**不对称的**（$g(\mu_i) \neq g(\mu_j)$ 时 $d_{ij}^2 \neq d_{ji}^2$）。但后续取 `torch.sqrt(d2)` 和 `mask` 时，不对称性被隐藏。对亲和矩阵而言，不对称意味着 $A_{ij} \neq A_{ji}$，违反直观。

**建议**：改为对称距离：

$$d_{ij}^2 = \frac{1}{2}(\mu_i - \mu_j)^\top \big[ g(\mu_i) + g(\mu_j) \big] (\mu_i - \mu_j)$$

或提案中的中点方案。实际差异可能很小，但在数学上更干净。

### 3.2 $\sigma$ 的选择

提案建议 $\sigma = \text{mean}(r_i)$。当前原子半径约 0.2~0.35，$\sigma \approx 0.25$。代入 $d_g$ 的指数衰减：

- $d_g = 0$ → $A = 1$
- $d_g = \sigma$ → $A = e^{-0.5} \approx 0.61$
- $d_g = 2\sigma$ → $A = e^{-2} \approx 0.14$
- $d_g = 3\sigma$ → $A = e^{-4.5} \approx 0.011$

这意味 $\tau_{\max} = 3\sigma$ 时，3σ 外的原子权重不到 1%，截断合理。但：

**需要检查当前场景中物体内部原子的平均测地距离**。如果物体较大，内部原子间距可能超过 0.75（即 3σ），导致物体内部被切分成不连通的子图，扩散无法传播。

**建议**：$\sigma$ 应该自适应于每个原子：

$$\sigma_i = \alpha \cdot \min_{j \neq i} d_g(\mu_i, \mu_j)$$

即每个原子的长度尺度由其最近邻居决定。这比全局 $\text{mean}(r_i)$ 更鲁棒。

### 3.3 $\alpha=0.5$, $T=2$ 的收敛行为

```python
for _ in range(T):
    F_new = (1 - α) * F_new + α * (S @ F_new)
```

这是一个**隐式欧拉积分**的显式近似，模拟 $\dot{F} = -L F$（$L = I - S$ 是图拉普拉斯），$T$ 步后特征趋向于图拉普拉斯的低频分量。

$T=2$ 时，初始特征保留 $(1-\alpha)^T = 0.25$，即 **75% 的信息来自邻居**。这个平滑强度相当高。对于当前 N≈100 的图，$T=2$ 的扩散半径约为 2 跳。

如果物体内部连通性好（最大路径长度约 5-10 跳），$T=2$ 不会导致全局坍缩。但如果物体内部有"瓶颈"（窄连接），$T$ 步数不够扩散越过瓶颈。

**建议**：$T$ 应该与特征图的直径相关，或采用**自适应停止条件**（当 $\|F^{(t+1)} - F^{(t)}\|_F < \epsilon$ 时停止）。

## 四、实施风险矩阵

| # | 风险 | 概率 | 影响 | 优先级 | 缓解方案 |
|---|---|---|---|---|---|
| R1 | 硬截断导致梯度截断，度量场在边界处无学习信号 | 高 | 中 | **P0** | 改用 sigmoid 软掩码 |
| R2 | $\sigma$ 过大 → 全局特征坍缩 | 中 | 高 | **P0** | 最近邻自适应 $\sigma_i$ + 保留 InfoNCE 安全网 |

## 六、第二阶段结论

### 可行但不神奇

特征扩散是一个**优雅但非决定性**的改进。它的价值不在于大幅提升 ARI，而在于：

1. **使特征空间平滑成为度量场学习的隐式监督**（扩散质量反映度量场准确度）
2. **在 3D 稀疏场景中提供额外的局部一致性信号**（缓解 InfoNCE 正样本不足）
3. **零额外参数，架构纯净**

ARI 从 0.175 → 0.5+ 的真正瓶颈预计仍在度量场分离锐利度和 InfoNCE 超参上，而非特征平滑度。

### 预计实际收益

| 指标 | 预期变化 | 确定性 |
|---|---|---|
| 簇内特征方差 | **↓ 30-50%** | 高（扩散直接作用于此） |
| ARI | ↑ 0.02-0.10 | 中（间接影响） |
| 收敛速度 | ↑ 10-20%（epoch 数降低） | 中低 |
| 3D 鲁棒性 | ↑ （待验证）| 低 |
