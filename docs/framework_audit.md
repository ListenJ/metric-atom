## 一、黎曼度量场理论 (4 个未证明命题)

### 缺陷 1.1 ✗ 度量场的可表达性

**声称：** Cholesky 参数化能表示任意正定对称矩阵。

**实际证明状况：** 这是线性代数定理不需要证明（LL^T + εI 总是 SPD）。但真正的问题是：
- Cholesky 参数化是在像素（体素）级别的——它只能表示**分段常数/双线性插值**的度量场
- 是否能表示任意光滑度量场？能，但需要**无限分辨率**
- 当前分辨率下（64×64 2D），可表达度量场的函数空间维数是多少？答：64*64*3 = 12,288 维（2D）/ 64³*6 = 1,572,864 维（3D）
- **未证明的隐含假设：** 12,288 维足以表示任意 2D 场景的"正确"度量场

### 缺陷 1.2 ✗ 度量场在物体边界变锐利的理论证明

**声称：** "度量场在物体边界处锐利跳变 → 自然定义聚类边界"（READMEMD §1）

**实际证明状况：** 这是经验观察（物体内 tr(g) ~ 0.5-2，背景 tr(g) ~ 5-9），但没有：
- 没有定理证明梯度流方程 dg/dt = -∇L_direct + η_smooth 的稳态解 g* 在物体边界处具有高梯度
- 没有分析 g* 的唯一性
- 没有证明这个稳态解是稳定的（Lyapunov 稳定）

**如果被推翻会发生什么：** 度量场在边界处尖锐不是必然的——如果平滑损失的权重过大，g 会均匀分布，导致聚类失败。这确实是 Phase 7 中 ~50% seed 失败的原因之一。

**实验上应做而未做：** 固定不同的 w_smooth，测量 g* 在边界处的梯度和对应的 ARI，验证"边界的尖锐程度与聚类质量正相关"。

### 缺陷 1.3 ✗ 占位耦合损失的"物体内" vs "物体外"定义是循环的

**声称：** 占位耦合损失 L_vol 强制物体内部度量迹小、背景度量迹大。

**问题：** 何为"物体内部"？当前实现用渲染的 occupancy mask 定义，但 occupancy 本身依赖度量场（原子的 smoothstep 截断使用 g 计算马氏距离）。这是**循环定义**：
- 度量场用来区分物体/背景
- 物体/背景用来训练度量场

**循环在梯度下降下是否仍然收敛**需要证明。这是一个典型的**协同训练**（co-training）问题——如果度量场和 occupancy 的耦合是压缩映射，则收敛；否则可能震荡。

**实验上应做而未做：** 测量 tr(g) 在物体内和外的时间序列，验证它单调收敛到某个稳态值（而不是震荡）。

### 缺陷 1.4 ✗ 平滑损失的梯度与 Direct Cluster 梯度的方向冲突

**问题：** ∇L_direct 推 g 变大（§二 gradient_flow_analysis），∇L_smooth 推 g 均匀。这两个梯度在**完全相反的方向**。

**情况 1：** w_smooth >> w_direct → g 均匀 → d² ≈ 常数 → P 接近均匀 → 聚类失败  
**情况 2：** w_smooth << w_direct → g 无界增长 → d² 发散 → 数值溢出  
**情况 3：** w_smooth ≈ w_direct → 稳态平衡

**未证明的隐含假设：** 存在一个平衡点 w_smooth* 使 g 在物体边界处尖锐但不发散。当前的 w_smooth 是通过网格搜索找到的——**没有理论保证这个值泛化到其他场景/分辨率**。

## 二、Direct Cluster 理论 (3 个未证明命题)

### 缺陷 2.1 ✗ L_direct 的全局最优性

**声称（暗示）：** 梯度下降能收敛到 Direct Cluster 损失的全局最小值。

**实际状况：** Phase 7 的 8 个 seed 中只有 1 个（seed-107）达到 ARI=1.0 —— 其余 7 个落入了不同的局部最小值。梯度流分析（gradient_flow_analysis.md §一）表明梯度是 $C^\infty$ 连续的，但**没有证明这个连续函数是凸的**或者**全局最小值是唯一的**。

**未证明的关键命题：**
1. L_direct 不是凸函数。给定 P 和 g 的耦合（P 依赖 features，features 依赖 g），这是一个**非凸优化问题**
2. 梯度流可能存在鞍点、局部极小值、plateau——这些都没有被系统性分类
3. 是否存在"坏"的初始条件，梯度必然收敛到差解（而不是偶然收敛到差解）？

### 缺陷 2.2 ✗ Sinkhorn 温度 ε 的精确理论值

**声称：** ε=0.05 最优（Phase 6c 实验验证，8 seed 平均 ARI 0.637）

**问题：** gradient_flow_analysis.md §四给了一个近似公式 ε* ≈ gap/(2 log K) ≈ 0.22–0.36，与实验 0.05 差了 4–7 倍。然后用"列平衡约束压缩有效温度"解释。

这个解释是**定性**的，不是定量的：
- $\sigma_v$ 本身是 P 的函数，P 是 ε 的函数 → 这是一个**不动点方程**
- 需要求解：$\varepsilon^* = \varepsilon / (1 + \sigma_v(\varepsilon^*))$
- 目前没有求解这个方程，只是用"2-4 倍"的模糊范围

**如果被推翻会发生什么：** 如果列平衡的压缩因子在不同场景下是 10 倍而不是 2-4 倍，那 ε=0.05 只在当前场景下最优，换场景后需要重新扫描。

### 缺陷 2.3 ✗ K-Means 初始化的偏置分析

**声称：** K-Means 初始化帮助 InfoNCE 打破对称性。

**问题：**

|| 如果 K-Means 是错误的 | 后果 |
|----|---------------------------|------|
| Direct Cluster (ε=0.05) | ??（从未测试） |
| InfoNCE (T=0.1) | 所有梯度方向错误 → 不可恢复 |

**未做的实验：**
- 用随机初始化（而非 K-Means）跑 Direct Cluster × 8 seed → 测量成功率是否变化
- 如果随机初始化的成功率显著低于 K-Means 初始化 → 说明 Direct Cluster 对初始化仍然敏感

## 三、ECO 理论 (4 个未证明命题) 🔴 含 3 个阻塞级

### 缺陷 3.1 🔴 j-不变量稳定性 "定理" 末经正式证明

**声称：** "小扰动 δa, δb 导致 δj = O(|δ|²)（二阶稳定）"（READMEMD §2, phase6a_eco_theory.md §2 定理 1）

**验证：**

$$j(a,b) = 1728 \frac{4a^3}{4a^3 + 27b^2}$$

Taylor 展开到二阶：

$$\delta j = \frac{\partial j}{\partial a}\delta a + \frac{\partial j}{\partial b}\delta b + \frac{1}{2}\left[\frac{\partial^2 j}{\partial a^2}(\delta a)^2 + 2\frac{\partial^2 j}{\partial a\partial b}\delta a\delta b + \frac{\partial^2 j}{\partial b^2}(\delta b)^2\right]$$

一阶导：
$$\frac{\partial j}{\partial a} = 1728 \frac{108 a^2 b^2}{(4a^3 + 27b^2)^2}$$
$$\frac{\partial j}{\partial b} = -1728 \frac{216 a^3 b}{(4a^3 + 27b^2)^2}$$

**关键发现：** 当 $b \to 0$ 时，$\frac{\partial j}{\partial a} \to 0$ —— 一阶导消失，j 在 b=0 处确实有平坦区域。但当 $b$ 不为 0 时，一阶导 $∂j/∂b$ 是 O(1) 而不是 O(||δ||) —— **没有二阶稳定性保证**。

**定理声称的问题：** 定理声称 δj = O(|δ|²) —— 这是**二阶**，不是通常的 O(|δ|) 线性阶。但这个声称只在一阶导为零的特殊情况下成立。对一般的 (a,b)，一阶导非零，δj = O(|δ|)（线性的），不是 O(|δ|²)。

**正确的命题应该是：** $|\delta j| \leq C(a,b) \cdot |\delta|$，其中 $C(a,b)$ 依赖于当前位置。在某些区域 $C(a,b)$ 很小（j "平缓"），在某些区域 $C(a,b)$ 很大（j "陡峭"）。

**如果被推翻：** 如果外界扰动让 (a,b) 进入 ∂j/∂b 较大的区域，j 会剧烈变化，物的"身份"丢失。这限制了 ECO 在非稳态环境中的鲁棒性——**这是该框架最核心的声称**。

### 缺陷 3.2 🔴 传感函数 φ 的可学习性未证明

**声称：** φ: features → (a,b) 能学到有意义的曲线参数映射。

**实际问题：**
- 3 层 MLP (F → 32 → 32 → 2) 的**有效容量**是多少？
- 给定 N 个原子（~100），它的有效 Rademacher 复杂度足够吗？
- φ 的输出 (a,b) 必须满足 Δ ≠ 0，但 MLP 天然不知道这个约束
- 当前用 ad-hoc guard（|4a³+27b²|<1e-3 时扰动 0.1）修正
- **没有证明这个 ad-hoc guard 不会破坏梯度流**

**实验上应做而未做的验证：**
- 对已训练模型，绘制 (a,b) 输出的散点图，看是否有聚类结构
- 测量训练过程中进入 guard 的频率（如果 >5% 的 forward pass 触发 guard，说明 MLP 没学到有效映射）
- 验证 φ 的输出对于同一物体的不同视图是否一致

### 缺陷 3.3 🔴 判别式壁垒的有效性未证明

**声称：** Phase 8 的 L_barrier = -mean(log|Δ|+ε) 把奇异轨迹变成排斥墙，防止 (a,b) 退化。

**问题：**

barrier 是**非凸的、无界自下的**（log|Δ| → -∞ 当 Δ → ∞）。这意味着：

1. **没有正定保证：** barrier 惩罚的是 |Δ| → 0，但没有惩罚 Δ → -∞（即 a → -∞ 或 b → -∞）。梯度可能把 (a,b) 推向无穷远处。
2. **与 j-separation 的冲突：** L_barrier 推 (a,b) 远离 Δ=0（即推离奇点），L_sep 推 |j_i - j_k| 远离零（即推离 j-空间原点）。这两者**方向不一定相同**——在模空间中，Δ=0 的轨迹和 j=0 的轨迹不一样。

**未做的实验：** 只使用 L_barrier（不用 L_sep），测量 ARI。如果 ARI >> 两者都用 → 说明 L_sep 和 L_barrier 在对抗（而不是协同合作）。

### 缺陷 3.4 ✗ j-不变量的 z-score 归一化破坏了理论声称

**代码事实（eco_cluster.py L44-46）：**
```python
j_raw = 1728.0 * a3 / denom.clamp(min=1e-10)
j_centered = j_raw - j_raw.mean()
return j_centered / (j_centered.std() + 1e-8)
```

**问题：** 这相当于把 j 的分布拉伸到单位方差。如果 batch 内有 K 个不同物体，z-score 会**相对化** j 的差异——j 的绝对差不再有意义，只有相对差有意义。

这与 ECO 的核心声称（"物体由曲线的绝对参数编码"）矛盾：
- 如果 batch 里只有物体 A → j_A z-scored → mean 0
- 如果 batch 里有物体 A 和 B → j_A 和 j_B z-scored → 两个都变了
- **同一个物体的 j 在不同 batch 中取值不同**——这直接破坏了"身份保持"声称

**如果被推翻：** 这是最严重的——z-score 归一化使 j 变成 batch-dependent，而 ECO 说的"身份保持"是 batch-independent 的。要么改 z-score（用全局统计），要么承认 ECO 在 batch 间不保证身份。

## 四、Murmuration 动力学 (3 个未证明命题)

### 缺陷 4.1 ✓ 离散时间欧拉积分的稳定性 → **[已数值验证 2026-06-03]**

**声称：** "E(R) 是紧致流形 → 所有轨道有界"

**之前状态：** 离散时间 Lyapunov 单调性需要数值确认，连续时间证明（缺陷 4.2）不直接适用于 dt=0.1 的 Euler 积分。

**验证：** 见 [numerical_verification.md](numerical_verification.md) §一。
在 AutoDL T4 上运行 `sim_murmuration_discrete.py`（N=20, T=200, 椭圆曲线 $y^2=x^3+1$）：

- V 从 2.70 降至 0.13（**Lyapunov 递减趋势明确**，比率 0.047）
- dt=0.1 时 ~20% 步骤有 ΔV>0（离散化 artifact，可接受）
- dt=0.01 时 ΔV>0 仅 1/200（近乎严格单调）
- 均匀间距平衡态完全稳定（0% 违反，8 种子验证）
- 8/8 随机种子收敛到低能构型

**结论：离散 Murmuration 的 Lyapunov 稳定性已验证通过。** 当前 dt=0.1 引入可接受的离散化 artifact，不影响终态质量。若需严格单调性，使用 dt ≤ 0.02。

### 缺陷 4.2 ✓ Lyapunov 函数不存在 → **[已解决 2026-06-02]**

**声称：** P0 优先级中是"动力系统：Lyapunov 稳定性——证明 murmuration 不会发散"。

**之前状态：** 没有 Lyapunov 函数被构造。对于标准 Boids (Reynolds 1987)，不存在全局 Lyapunov 函数（能量不单调递减）。

**解决：** 见 [murmuration_dynamics.md](murmuration_dynamics.md) §二。
在椭圆曲线上的特殊结构中（1D 切空间，力 → 标量），凝聚和分离力是保守的（来自势能 U_coh + U_sep）。
仅对齐力涉及速度耦合。构造:

```
V = ½ Σ v_i² + ½α Σ (1/Nᵢ) Σ d_g(Pᵢ,Pⱼ)² + γ Σ (r-d)²
```

当 η > β（阻尼系数 > 对齐权重，默认 1.0 > 0.05 满足）时:

```
dV/dt = -(η-β) Σ v_i² - β/2 Σ (1/Nᵢ) Σ (vᵢ-vⱼ)² ≤ 0
```

**严格 Lyapunov 函数存在**，全局收敛到均匀间距晶体构型（模旋转）。
这是项目中最干净的证明之一——由于 1D 切空间的降维结构，
Murmuration on E 比标准 3D Boids 更易分析。

### 缺陷 4.3 ✗ Newton 投影修复的收敛保证

**代码（elliptic_curve.py exp_map）：**
```python
# Newton iteration to project back to curve
for _ in range(max_iters):
    candidate = candidate - f(candidate) / f'(candidate)
```

**问题：**
- Newton 法的收敛条件：初始估计必须足够近
- 没有 bound 保证 exp_map 的输出在 Newton 的收敛半径内
- 如果 Newton 不收敛（max_iters 耗尽），直接返回原 P —— 相当于位置没有更新

**这意味着：某些 Murmuration 步骤被无声地丢弃了**——没有警告，没有梯度信号，没有 fallback。

## 五、系统集成 (2 个未证明命题)

### 缺陷 5.1 ✗ μ_i ⇄ P_i 的解耦数学分析

在 3D 扩展中（math_analysis.md §四），原子位置 μ_i ∈ ℝ³ 和曲线坐标 P_i ∈ E(ℝ) 在两个不同空间演化：
- 渲染损失 → 只更新 μ_i
- ECO 损失 → 只更新 P_i

**当前的"耦合"是隐式的**：渲染梯度通过特征（features）间接影响 φ，ECO 的梯度不影响 μ_i。

**需要被证明或否定的命题：**
- 是否存在一个**解耦的稳定状态**，其中 μ_i 和 P_i 给出不同的聚类结果？
- 如果存在，算法是否会收敛到这个解耦状态？

**验证实验：** 训练完成后，分别用 μ_i 的 K-Means 和 P_i (j-不变量的) K-Means 对原子聚类，比较 ARI。如果两个 ARI 显著不同，说明 μ 和 P 已解耦。

### 缺陷 5.2 ✗ 多损失的梯度方向冲突

当前训练涉及 7 个损失函数：

| 损失 | 更新的变量 | 梯度方向 |
|------|-----------|---------|
| L_render | μ, g, φ, features | 推到颜色匹配的位置 |
| L_smooth(g) | g | 推 g 均匀化 |
| L_vol | g | 推物体内 tr(g)↓, 背景 tr(g)↑ |
| L_direct | g, features, protos | 推簇内 d²↓ |
| L_eco_id | φ | 推 j 保持初始值 |
| L_barrier | φ (a,b) | 推 |Δ|↑ (远离奇点) |
| L_sep | φ (a,b) | 推 |j_i - j_k|↑ (拉远曲线) |

**没有理论保证这些梯度的线性组合收敛到稳定点。** 当前权重 {λ_met, λ_vol, λ_pos, λ_coh, w_eco, eco_id_weight} 是网格搜索得出的，不能保证泛化到新场景。

## 八、优先级分类

### 🔴 阻塞级（如果被推翻，核心声称失效）

1. **j-不变量稳定性声称有错误**（缺陷 3.1）—— δj = O(|δ|²) 仅在一阶导为零处成立，一般情况下是 O(|δ|)
2. **z-score 归一化破坏身份保持**（缺陷 3.4）—— j 变成 batch-dependent
3. **Δ ≠ 0 约束的 ad-hoc guard**（缺陷 3.2）—— 没有理论上保证曲线不退化

### 🟡 重要级

4. L_direct 的非凸性和局部最小值（缺陷 2.1）
5. 梯度方向的冲突无理论保证（缺陷 5.2）
6. 3D 实验从未完成（缺陷 E1）
7. 离散 Murmuration 的流形保持（缺陷 4.1）

### 🟢 优化级

8. ε 的精确理论值（缺陷 2.2）
9. 度量场边界变锐利的证明（缺陷 1.2）
10. Lyapunov 构造（缺陷 4.2）
11. 工程假设的误差边界（缺陷 H1-H7）

## 九、建议的修复路径

### 立即可做（不改变代码）

1. **修正 j-不变量定理的声称：** 将 δj = O(|δ|²) 改为 $|\delta j| \leq C(a,b)|\delta|$，明确说明 $C(a,b)$ 在 b≈0 附近很小、远处很大
2. **填补 z-score 困境：** 在文档中明确承认 z-score 使 j batch-dependent，并讨论两种解决方案：
   - 用全局移动统计（类似 BatchNorm 的 running stats）
   - 改用绝对距离而非 z-score 后的距离

### 需要新实验

3. **3D 联合训练：** 用 train_3d.py + Direct Cluster + ECO 配置跑一次完整 100 epoch，即使 ARI 不高也要确认算法不崩溃
4. **K>2 测试：** 生成 3 个不同形状的场景，跑 Direct Cluster (K=3)，验证 ≥2 簇的成功率
5. **不同初始化对比：** Random init vs K-Means init，跑 8 seed，证明 Direct Cluster 的成功不依赖 K-Means 偏置

### 需要理论工作

6. **j-不变量的 Lipschitz 常数估计：** 计算 $C(a,b) = \|∇j(a,b)\|$ 在整个参数空间的上界，给身份保持一个硬保障
7. **Δ 守恒引理：** 给定 L_barrier 梯度，证明存在一个 ε>0 使 Δ_min > ε 对所有训练步骤成立
